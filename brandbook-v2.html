Раздел 01

Цветовая палитра

База осталась «издательской» — белый фон, глубокий синий, много воздуха. Но статичный небесно-голубой заменён на энергичный электрик-синий, а вместо малинового — коралловый: цвет пульса, финишной ленты и сегодняшней цели.

Раздел 02

Типографика

Всё без засечек. Archivo — атлетичный, плотный гротеск для заголовков и крупных цифр; наклонное начертание (italic) даёт визуальное ощущение движения. Manrope — дружелюбный и очень читаемый — для текста, кнопок, таблиц. IBM Plex Mono — для темпа, времени и любых «приборных» значений: моноширинные цифры не прыгают при обновлении в реальном времени.

Раздел 04

Кнопки

Manrope 800, скругление 8px — удобно для пальца. Движение — в микровзаимодействиях: при наведении кнопка приподнимается на 1px, а стрелка в кнопке «вперёд» сдвигается вправо.

Primary. Electric Blue → темнее при наведении + подъём 1px. Главное действие, одна на экран.
Go / Accent. Коралловая, со стрелкой, которая сдвигается при наведении. Только «моменты старта»: начать тренировку, принять план, зарегистрироваться на старт.
Secondary. Белая с синей рамкой 1.5px. Альтернативные действия рядом с Primary.
Tertiary. Серая панель без рамки. Нейтральные действия: отмена, назад, «позже».
Link. Текстовая, Electric Blue 800. Второстепенные переходы.
Disabled. Panel-фон, серый текст, без подъёма и тени. Одинаково для всех типов.

Раздел 06

Раздел 07

Цитаты и панели

Мотивационные цитаты — капсом, наклонно, с коралловым смысловым ядром. Панели — три уровня: серая (нейтральная), голубая (выделение), navy (главная цель).

“Consistency beats intensity — but today you need both.”

Coach Insight | Еженедельный отчёт

Серая панель

Стандартная подложка для группировки контента. Радиус 10px, без рамки и тени.

Голубая панель

Подсказки тренера, «что нового», выбранное состояние в списках.

Navy-панель · Цель

Инверсная — для главной цели сезона. Заголовок внутри — коралловый.

Раздел 08

Правила

✓ Делаем

  • Тема всегда светлая, много воздуха, отступы кратны 8px
  • Archivo italic — только большие цифры и hero-заголовки
  • Живые метрики (темп/пульс/время) — Plex Mono, tabular-nums
  • Коралловый = импульс: 1–2 элемента на экран
  • Цвета дисциплин — только для маркировки видов спорта
  • Движение через микровзаимодействия: подъём кнопки, сдвиг стрелки

✗ Не делаем

  • Шрифты с засечками — их в системе больше нет
  • Наклон/капс в основном тексте и таблицах
  • Коралловый в фоновых заливках больших областей
  • Больше 3 серий базовой палитры на одном графике
  • Агрессивные анимации: параллаксы, вращения, мигания
  • Чистый чёрный #000 — только Ink #1E2530
Раздел 09

CSS-токены

Готовый блок для разработчика. Шрифты — Google Fonts, бесплатные: Archivo (700–900 + italic), Manrope (400–800), IBM Plex Mono (400–600).

:root {
  /* Базовые цвета */
  --c-blue:       #2F6DE0;   /* действия, ссылки, данные */
  --c-navy:       #14294B;   /* заголовки, цифры */
  --c-coral:      #F1553F;   /* энергия: 1–2 шт. на экран */
  --c-ink:        #1E2530;
  --c-ink-soft:   #5C6675;
  --c-panel:      #F3F5F8;
  --c-panel-blue: #EAF1FC;
  --c-line:       #DCE2EA;
  --c-bg:         #FFFFFF;

  /* Дисциплины */
  --c-swim: #35A6D9;  --c-bike: #F2A33C;  --c-run: #EF5B5B;

  /* Тренировочные зоны */
  --z1: #9DB8E8; --z2: #5E8BE0; --z3: #2F6DE0;
  --z4: #F2A33C; --z5: #F1553F;

  /* Шрифты */
  --f-display: 'Archivo', sans-serif;      /* 800–900, italic для цифр */
  --f-body:    'Manrope', sans-serif;      /* 400–800 */
  --f-mono:    'IBM Plex Mono', monospace; /* темп, время, метрики */

  /* Шкала */
  --fs-hero: 64px; --fs-h1: 38px; --fs-h2: 26px;
  --fs-h3: 19px;   --fs-body: 16px; --fs-small: 13px; --fs-micro: 11px;

  /* Форма и движение */
  --radius: 10px;  --radius-btn: 8px;  --space: 8px;
  --skew: -8deg;   /* наклон «скорости» */
  --shadow: 0 2px 10px rgba(20,41,75,.07);
}
